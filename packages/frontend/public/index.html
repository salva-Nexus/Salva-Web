SALVA

On-Chain Name Service & Payment Protocol for Nigeria

Salva is Nigeria's on-chain name service and financial protocol, currently deployed on Base Testnet. It maps human-readable name aliases to wallet addresses and account numbers — so you send payments to a name like charles@salva or sandra_eberechi@uba, not a long hexadecimal address.

Salva is wallet-agnostic. Register your alias from Salva Wallet, Coinbase, MetaMask, or your bank app. Each namespace (@salva, @coinbase, @uba) is isolated — identities across registries never collide.

How it works

  • Create or connect any compatible wallet
  • Register a name alias — "charles@salva" or "aggregatorv3_eth@chainlink"
  • Your alias maps to your wallet address or account number on-chain
  • Send and receive crypto or NGNs (Nigerian Naira stablecoin) by name
  • Multiple registries (namespaces) each have their own isolated identity space
  • Currently on Base Testnet — expanding to more chains

Why Salva

Crypto payments today require copying 42-character wallet addresses. Salva solves this with a permanent, phishing-resistant on-chain name registry that links your identity to your wallet — the same way DNS links domain names to IP addresses, but for blockchain.

Salva Smart Wallet

Salva also ships its own Smart Wallet powered by Safe Protocol — no seed phrases, zero gas fees for users, gasless NGNs transfers. Built for everyday Nigerians.
